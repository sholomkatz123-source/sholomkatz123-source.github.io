# Neileich Website

Static, multi-page site for Neileich.

## Structure
- `index.html` — Home
- `programs.html` — Programs
- `impact.html` — Impact
- `leadership.html` — Leadership
- `contact.html` — Contact
- `donate.html` — Donate (redirects to Charidy)
- `assets/style.css` — Shared styles
- `assets/main.js` — Shared JS

## Local Preview
Open any HTML file in your browser. No build steps required.

## Deploy on GitHub Pages
1. Push to a public repo.
2. Settings → Pages → **Deploy from branch**, branch: `main`, folder: `/ (root)`.
3. Your site will be live shortly.
